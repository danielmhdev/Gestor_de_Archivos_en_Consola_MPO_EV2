# Gestor de Archivos en Consola

Este es un proyecto desarrollado para la segunda evaluación de MPO. Se trata de una aplicación de consola escrita en Python que permite gestionar archivos de manera sencilla desde la terminal.

## Descripción

El programa ofrece funcionalidades básicas para trabajar con archivos y directorios directamente desde la línea de comandos, facilitando tareas comunes de administración de archivos.

## Requisitos

- Python 3.x instalado en tu sistema

## Uso

Para ejecutar el programa, abre una terminal en la carpeta del proyecto y ejecuta:

`python main.py`

## Funcionalidades

El gestor permite realizar operaciones típicas de manejo de archivos a través de un menú interactivo en la consola.
